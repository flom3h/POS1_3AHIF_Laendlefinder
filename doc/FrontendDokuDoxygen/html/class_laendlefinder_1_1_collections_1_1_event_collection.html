Ausführliche Beschreibung

Repräsentiert eine Sammlung von Events, die ObservableCollection erweitert. Diese Klasse ermöglicht das Serialisieren von Events in eine JSON-Datei, das Zeichnen der Events in ein Panel, sowie das Suchen und Filtern von Events basierend auf verschiedenen Kriterien.
