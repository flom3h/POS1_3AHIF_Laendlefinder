\doxysubsection{Ausführliche Beschreibung}
More\+Info\+Page. 

Repräsentiert die Detailseite eines Events. Zeigt Informationen zu einem Event an, einschließlich Name, Datum, Uhrzeit, Ort, Beschreibung und Bild. Ermöglicht das Hinzufügen oder Entfernen des Events aus den Favoriten.
